# 🦸‍♂️ 构建思维框架 (thinking-framework-skills)

## 🌟 说明 (Description)
这是一个用于构建和完善用户思维框架的 Skill，帮助用户结构化地思考问题并提炼核心逻辑。

## 🎯 目标 (Objectives)
- 帮助用户理清思路
- 构建系统化的心智模型
- 将散乱的信息结构化
- 提炼核心逻辑和概念

## 🛠 功能 (Features)
- 分析复杂信息并建立概念之间的联系
- 通过图表等形式构建心智框架
- 输出结构化的分析报告

## 📁 目录结构 (Directory Structure)
- `scripts/` - 包含可执行脚本
- `references/` - 包含参考资料
- `store/` - 包含技能的相关存储
